# SOMA — Local AI Personal Assistant
### iOS Native Architecture & Build Plan for Claude Code

> **North Star**: Every byte of user data stays on-device. No telemetry, no cloud inference,
> no analytics. The network is only ever used for the initial Gemma 4 model download.

---

## 0. What We Are Building

A fully local personal AI assistant for iPhone, built entirely in Swift with SwiftUI.
The app ingests data from the user's calendar, contacts, notes, and manually exported
app data (WhatsApp, mail), maintains a private on-device knowledge base with semantic
RAG retrieval, and runs Gemma 4 on-device via Google's AI Edge SDK to answer questions,
summarise daily context, and act as a proactive personal assistant.

**No Kotlin. No cross-platform. No bridges. Pure Swift, pure iOS, pure local.**

**Core user loop:**
1. Open app → morning digest from calendar + recent notes, generated by Gemma 4
2. Chat with assistant → Gemma 4 answers using retrieved knowledge base context
3. Voice-log a thought → on-device Speech framework transcribes, stored in KB
4. Share content from any app → Share Extension ingests into KB automatically
5. Import exports (WhatsApp .txt, mail .mbox) → parsed and stored in KB

---

## 1. Core Principles

- **Locality always.** No feature may send user data off-device. Period.
- **Native first.** Use Apple frameworks wherever they exist. Less code, better performance.
- **Swift Concurrency throughout.** `async/await` and `Actor` for all concurrent work.
- **SwiftData for persistence.** iOS 17+ only — modern, Swift-native, no boilerplate.
- **CoreML for embeddings.** MiniLM runs on the Neural Engine. No third-party ML runtime needed for this.
- **LiteRT-LM for Gemma 4.** Google's official Swift SDK — clean API, no C interop required.
- **Graceful degradation.** App stays usable for KB browsing even while model loads.

```swift
        self.content = content

        self.tags = tags
```

```swift
// Gemma 4 instruction-tuned chat format.
// Reference: https://ai.google.dev/gemma/docs/formatting
// Every deviation from this format produces garbage output — always use these methods.

enum PromptTemplates {

    static let systemPrompt = """
    You are Soma, a private local AI assistant. You have access to the user's personal \
    knowledge base — notes, calendar events, contacts, and imported data from apps. \
    Everything runs on-device. Be concise, personal, and genuinely useful. \
    When you don't know something, say so clearly. Never fabricate facts about the user.
    """

    static let toolSystemPrompt = """
    You are Soma. You can call tools to retrieve information or take actions. \
    When calling a tool, respond ONLY with a JSON object in this exact format — no other text:
    {"tool": "tool_name", "parameters": {"key": "value"}}
    After receiving the tool result, respond naturally in plain text.
    """

    static func chat(
        history: [ConversationMessage],
        userMessage: String,
        retrievedContext: String
    ) -> String {
        var prompt = "<start_of_turn>system\n\(systemPrompt)"
        if !retrievedContext.isEmpty {
            prompt += "\n\n## Relevant context from your knowledge base:\n\(retrievedContext)"
        }
        prompt += "<end_of_turn>\n"

        for message in history.suffix(10) {
            let role = message.role == "user" ? "user" : "model"
            prompt += "<start_of_turn>\(role)\n\(message.content)<end_of_turn>\n"
        }

        prompt += "<start_of_turn>user\n\(userMessage)<end_of_turn>\n"
        prompt += "<start_of_turn>model\n"
        return prompt
    }

    static func toolDetection(
        userMessage: String,
        availableSkills: [any Skill],
        context: String
    ) -> String {
        let skillList = availableSkills.map {
            "- \($0.name): \($0.description)\n  Parameters: \($0.parametersSchema)"
        }.joined(separator: "\n")

        return """
        <start_of_turn>system
        \(toolSystemPrompt)

        Available tools:
        \(skillList)
        \(context.isEmpty ? "" : "\nContext:\n\(context)")
        <end_of_turn>
        <start_of_turn>user
        \(userMessage)<end_of_turn>
        <start_of_turn>model

        """
    }

    static func digest(events: String, notes: String, dateString: String) -> String {
        """
        <start_of_turn>system
        Generate a concise morning briefing. Format: 3–5 bullet points. \
        Be specific, not generic. Lead with the most time-sensitive item. \
        Use plain language, no markdown headers.
        <end_of_turn>
        <start_of_turn>user
        Date: \(dateString)

        Calendar events:
        \(events.isEmpty ? "No events today." : events)

        Recent notes:
        \(notes.isEmpty ? "No recent notes." : notes)
        <end_of_turn>
        <start_of_turn>model

        """
    }
}
```

### 5.4 Model Downloader

```swift
import Foundation

@Observable
final class ModelDownloader {

    var progress: Double = 0
    var isDownloading = false
    var error: String?

    static let modelsDirectory: URL = {
        FileManager.default
            .urls(for: .documentDirectory, in: .userDomainMask)[0]
            .appendingPathComponent("Models", isDirectory: true)
    }()

    func modelPath(for variant: GemmaVariant) -> URL? {
        let url = Self.modelsDirectory.appendingPathComponent(variant.rawValue)
        return FileManager.default.fileExists(atPath: url.path) ? url : nil
    }

    func download(variant: GemmaVariant) async {
        isDownloading = true; progress = 0; error = nil
        try? FileManager.default.createDirectory(at: Self.modelsDirectory, withIntermediateDirectories: true)
        let destination = Self.modelsDirectory.appendingPathComponent(variant.rawValue)

        let config = URLSessionConfiguration.default
        config.timeoutIntervalForResource = 7200  // 2 hours for large files

        do {
            let (asyncBytes, response) = try await URLSession(configuration: config)
                .bytes(from: variant.downloadURL)
            guard (response as? HTTPURLResponse)?.statusCode == 200 else {
                throw URLError(.badServerResponse)
            }

            let total = Double(response.expectedContentLength)
            var received: Double = 0

            FileManager.default.createFile(atPath: destination.path, contents: nil)
            let handle = try FileHandle(forWritingTo: destination)
            defer { try? handle.close() }

            var buffer = Data(capacity: 1_048_576)
            for try await byte in asyncBytes {
                buffer.append(byte)
                received += 1
                if buffer.count >= 1_048_576 {
                    handle.write(buffer)
                    buffer.removeAll(keepingCapacity: true)
                    if total > 0 { await MainActor.run { progress = received / total } }
                }
            }
            if !buffer.isEmpty { handle.write(buffer) }
            await MainActor.run { progress = 1.0; isDownloading = false }

        } catch {
            try? FileManager.default.removeItem(at: destination)
            await MainActor.run { self.error = error.localizedDescription; isDownloading = false }
        }
    }
}
```

```swift
import CoreML
import Accelerate
import Foundation

// MiniLM-L6-v2 produces 384-dimensional sentence embeddings.
//
// SETUP — convert ONNX to CoreML before building:
//   pip install coremltools onnx transformers
//   python convert_minilm.py
//   (see convert_minilm.py script at bottom of this section)
//   Then drag MiniLM.mlpackage into the Xcode project, target: Soma

actor EmbeddingService {

    private var model: MLModel?
    private(set) var isLoaded = false
    static let dimensions = 384

    func load() throws {
        let config = MLModelConfiguration()
        config.computeUnits = .all   // Uses Neural Engine when available
        guard let url = Bundle.main.url(forResource: "MiniLM", withExtension: "mlpackage") else {
            throw EmbeddingError.modelNotFound
        }
        model = try MLModel(contentsOf: url, configuration: config)
        isLoaded = true
    }

    func embed(_ text: String) async throws -> [Float] {
        guard let model else { throw EmbeddingError.notLoaded }
        // Claude Code: after converting MiniLM, run the model to discover input/output keys:
        //   print(model.modelDescription.inputDescriptionsByName.keys)
        //   print(model.modelDescription.outputDescriptionsByName.keys)
        // Then replace "input_ids", "attention_mask", "sentence_embedding" below with actual keys.

        let tokens = tokenize(preprocess(text), maxLength: 128)
        let inputIDs = try MLMultiArray(shape: [1, 128], dataType: .int32)
        let attMask = try MLMultiArray(shape: [1, 128], dataType: .int32)
        for i in 0..<128 {
            inputIDs[i] = NSNumber(value: i < tokens.count ? tokens[i] : 0)
            attMask[i] = NSNumber(value: i < tokens.count ? 1 : 0)
        }
        let input = try MLDictionaryFeatureProvider(dictionary: [
            "input_ids": MLFeatureValue(multiArray: inputIDs),
            "attention_mask": MLFeatureValue(multiArray: attMask)
        ])
        let output = try await model.prediction(from: input)
        guard let arr = output.featureValue(for: "sentence_embedding")?.multiArrayValue else {
            throw EmbeddingError.badOutput
        }
        var vector = [Float](repeating: 0, count: Self.dimensions)
        for i in 0..<Self.dimensions { vector[i] = arr[i].floatValue }
        return normalise(vector)
    }

    private func normalise(_ v: [Float]) -> [Float] {
        var norm: Float = 0
        vDSP_svesq(v, 1, &norm, vDSP_Length(v.count))
        norm = sqrtf(norm)
        guard norm > 0 else { return v }
        var result = [Float](repeating: 0, count: v.count)
        vDSP_vsdiv(v, 1, &norm, &result, 1, vDSP_Length(v.count))
        return result
    }

    private func preprocess(_ text: String) -> String {
        text.replacingOccurrences(of: "\\s+", with: " ", options: .regularExpression)
            .trimmingCharacters(in: .whitespacesAndNewlines)
            .prefix(1000).description
    }

    // Minimal whitespace tokenizer for v1.
    // Claude Code: replace with a proper WordPiece tokenizer for production.
    // Reference: https://github.com/hollance/BERT-CoreML
    private func tokenize(_ text: String, maxLength: Int) -> [Int] {
        var ids = [101]  // [CLS]
        for word in text.lowercased().components(separatedBy: .whitespacesAndNewlines) {
            ids.append(contentsOf: word.unicodeScalars.prefix(5).map { Int($0.value) % 28000 + 1000 })
            if ids.count >= maxLength - 1 { break }
        }
        ids.append(102)  // [SEP]
        return ids
    }
}
```

```swift
import Accelerate
import Foundation

// In-memory vector index. Loaded from SwiftData on launch. All searches use
// Accelerate vDSP for dot-product on L2-normalised vectors (= cosine similarity).

actor VectorStore {

    struct Entry {
        let embeddingID: UUID
        let entryID: UUID
        let chunkText: String
        let vector: [Float]
    }

    private var entries: [Entry] = []

    func load(_ records: [(embeddingID: UUID, entryID: UUID, chunkText: String, vector: [Float])]) {
        entries = records.map { Entry(embeddingID: $0.embeddingID, entryID: $0.entryID, chunkText: $0.chunkText, vector: $0.vector) }
    }

    func add(embeddingID: UUID, entryID: UUID, chunkText: String, vector: [Float]) {
        entries.append(Entry(embeddingID: embeddingID, entryID: entryID, chunkText: chunkText, vector: vector))
    }

    func remove(entryID: UUID) {
        entries.removeAll { $0.entryID == entryID }
    }

    func topK(query: [Float], k: Int) -> [(entryID: UUID, score: Float, chunkText: String)] {
        guard !entries.isEmpty else { return [] }
        let dim = vDSP_Length(query.count)
        let scores: [(Entry, Float)] = entries.map { entry in
            var dot: Float = 0
            vDSP_dotpr(query, 1, entry.vector, 1, &dot, dim)
            return (entry, dot)
        }
        return scores.sorted { $0.1 > $1.1 }.prefix(k)
            .map { (entry, score) in (entryID: entry.entryID, score: score, chunkText: entry.chunkText) }
    }

    var count: Int { entries.count }
}
```

```swift
struct TextChunker {
    let chunkSize: Int
    let stride: Int

    init(chunkSize: Int = 500, stride: Int = 400) {
        self.chunkSize = chunkSize
        self.stride = stride
    }

    func chunk(_ text: String) -> [String] {
        guard text.count > chunkSize else { return [text] }
        var chunks: [String] = []
        var start = text.startIndex
        while start < text.endIndex {
            let end = text.index(start, offsetBy: chunkSize, limitedBy: text.endIndex) ?? text.endIndex
            chunks.append(String(text[start..<end]))
            guard let next = text.index(start, offsetBy: stride, limitedBy: text.endIndex) else { break }
            start = next
        }
        return chunks
    }
}
```

```swift
            return try await semanticSearch(query: query, topK: topK)
        }

    func recent(source: EntrySource? = nil, limit: Int = 20) throws -> [KnowledgeEntry] {

            sortBy: [SortDescriptor(\.createdAt, order: .reverse)]
        )

        descriptor.fetchLimit = limit

    }

        modelContext.delete(entry)
        try modelContext.save()
    }

    func loadVectorStoreFromDB() async {
        guard let entries = try? modelContext.fetch(FetchDescriptor<KnowledgeEntry>()) else { return }
        let records = entries.flatMap { entry in
            entry.embeddings.map { rec in (embeddingID: rec.id, entryID: entry.id, chunkText: rec.chunkText, vector: rec.floatVector) }
        }
        await vectorStore.load(records)
    }

    // MARK: Private

    private func embedEntry(_ entry: KnowledgeEntry) async throws {

        for (i, chunk) in chunks.enumerated() {
            let vector = try await embeddingService.embed(chunk)
            await MainActor.run {
                let rec = EmbeddingRecord(chunkIndex: i, chunkText: chunk, vector: vector)
                entry.embeddings.append(rec)
                try? modelContext.save()
            }
            await vectorStore.add(embeddingID: UUID(), entryID: entry.id, chunkText: chunk, vector: vector)
        }
    }

    private func semanticSearch(query: String, topK: Int) async throws -> [KnowledgeEntry] {
        let queryVec = try await embeddingService.embed(query)
        let results = await vectorStore.topK(query: queryVec, k: topK)
        var seen = Set<UUID>()
        return results.compactMap { result -> KnowledgeEntry? in
            guard !seen.contains(result.entryID) else { return nil }
            seen.insert(result.entryID)
            let id = result.entryID
            let descriptor = FetchDescriptor<KnowledgeEntry>(predicate: #Predicate { $0.id == id })
            return try? modelContext.fetch(descriptor).first
        }

    private func keywordSearch(query: String, limit: Int) -> [KnowledgeEntry] {
        let q = query.lowercased()
        var descriptor = FetchDescriptor<KnowledgeEntry>(
            predicate: #Predicate { $0.content.localizedStandardContains(q) },
            sortBy: [SortDescriptor(\.createdAt, order: .reverse)]
        )
        descriptor.fetchLimit = limit
        return (try? modelContext.fetch(descriptor)) ?? []
    }
```

```swift
import Foundation

final class RAGPipeline {

    private let gemma: GemmaSession
    private let knowledgeRepo: KnowledgeRepository
    private let skillRegistry: SkillRegistry
    private let parser = SkillParser()

    init(gemma: GemmaSession, knowledgeRepo: KnowledgeRepository, skillRegistry: SkillRegistry) {
        self.gemma = gemma
        self.knowledgeRepo = knowledgeRepo
        self.skillRegistry = skillRegistry
    }

    func chat(userMessage: String, history: [ConversationMessage]) -> AsyncThrowingStream<String, Error> {
        AsyncThrowingStream { continuation in
            Task {
                do {
                    // 1. Retrieve relevant KB context
                    let retrieved = (try? await knowledgeRepo.search(query: userMessage, topK: 5)) ?? []
                    let context = retrieved.map { entry in
                        "[\(entry.entrySource.rawValue.uppercased()) · \(entry.createdAt.formatted(date: .abbreviated, time: .omitted))]\n\(entry.content)"
                    }.joined(separator: "\n\n---\n\n")

                    // 2. Skill detection — non-streaming, fast
                    var augmentedContext = context
                    let toolPrompt = PromptTemplates.toolDetection(
                        userMessage: userMessage,
                        availableSkills: skillRegistry.enabledSkills,
                        context: context
                    )
                    if let rawResponse = try? await gemma.generateBlocking(prompt: toolPrompt),
                       let call = parser.parse(rawResponse) {
                        let result = await skillRegistry.dispatch(call)
                        augmentedContext = result.content + (context.isEmpty ? "" : "\n\n" + context)
                    }

                    // 3. Stream chat response
                    let chatPrompt = PromptTemplates.chat(
                        history: history,
                        userMessage: userMessage,
                        retrievedContext: augmentedContext
                    )
                    for try await token in await gemma.generate(prompt: chatPrompt) {
                        continuation.yield(token)
                    }
                    continuation.finish()
                } catch {
                    continuation.finish(throwing: error)
                }
            }
        }
    }
}
```

```swift
struct AnyCodable: Codable, @unchecked Sendable {
    let value: Any
    var stringValue: String? { value as? String }
    var intValue: Int? { value as? Int }
    var doubleValue: Double? { value as? Double }
    var boolValue: Bool? { value as? Bool }
    var arrayValue: [AnyCodable]? { value as? [AnyCodable] }

    init(_ value: Any) { self.value = value }

    init(from decoder: Decoder) throws {
        let c = try decoder.singleValueContainer()
        if let s = try? c.decode(String.self) { value = s }
        else if let i = try? c.decode(Int.self) { value = i }
        else if let d = try? c.decode(Double.self) { value = d }
        else if let b = try? c.decode(Bool.self) { value = b }
        else if let a = try? c.decode([AnyCodable].self) { value = a }
        else { value = "" }
    }

    func encode(to encoder: Encoder) throws {
        var c = encoder.singleValueContainer()
        switch value {
        case let s as String: try c.encode(s)
        case let i as Int: try c.encode(i)
        case let d as Double: try c.encode(d)
        case let b as Bool: try c.encode(b)
        default: try c.encode("")
        }
    }
```

## 14. UI Screens (SwiftUI)

File: `Soma/UI/Chat/ChatViewModel.swift`

```swift
import Foundation

@Observable
final class ChatViewModel {
    var messages: [ConversationMessage] = []
    var streamingBuffer = ""
    var isGenerating = false
    var errorMessage: String?
    private let pipeline: RAGPipeline

    init(pipeline: RAGPipeline) { self.pipeline = pipeline }

    func send(_ text: String) {
        guard !text.trimmingCharacters(in: .whitespaces).isEmpty else { return }
        messages.append(ConversationMessage(role: "user", content: text))
        isGenerating = true; streamingBuffer = ""
        Task {
            do {
                for try await token in pipeline.chat(userMessage: text, history: messages.dropLast()) {
                    streamingBuffer += token
                }
                messages.append(ConversationMessage(role: "assistant", content: streamingBuffer))
                streamingBuffer = ""
            } catch { errorMessage = error.localizedDescription }
            isGenerating = false
        }
    }
}
```

## 18. Notes for Claude Code

1. **Always open `Soma.xcworkspace`**, not `Soma.xcodeproj`. The CocoaPods workspace
   is required. Opening `.xcodeproj` directly will fail to resolve MediaPipeTasksGenAI.

2. **Verify MediaPipe SDK API before writing GemmaSession.** After `pod install`, run:
   ```bash
   find Pods/MediaPipeTasksGenAI -name "*.h" | xargs grep -l "generateResponse"
   ```
   Read the header. The exact class name, method signatures, and options struct fields
   must come from the installed SDK — not from memory or online examples, which may lag
   the current version. Documentation:
   https://developers.google.com/mediapipe/solutions/genai/llm_inference/ios

3. **SwiftData `@MainActor` requirement.** `ModelContext` and all `@Model` fetch/save
   operations must run on the main actor. `KnowledgeRepository` is `@MainActor` for
   this reason. Background work (embedding) uses `Task.detached` and calls back to
   `MainActor.run { }` for any model mutation.

4. **`@Observable` requires iOS 17+.** This is intentional and non-negotiable.
   Do not add `ObservableObject` fallbacks or `@Published` properties.

5. **EmbeddingRecord vector as Data.** The `floatVector` computed property converts
   `Data` → `[Float]` via `bindMemory`. This is the correct approach — do not use
   JSON encoding for float arrays (10x larger, 50x slower).

6. **App Group entitlement must be set in Xcode UI.** The `.entitlements` file alone
   is not sufficient — Xcode must register the group with the provisioning system.
   Steps: Target → Signing & Capabilities → + Capability → App Groups →
   add `group.com.soma.shared`. Do this for both `Soma` and `ShareExtension` targets.

7. **CoreML input/output keys.** After `convert_minilm.py`, inspect the model:
   ```python
   import coremltools as ct
   m = ct.models.MLModel('MiniLM.mlpackage')
   print(m.input_description)
   print(m.output_description)
   ```
   Update `EmbeddingService.embed()` dictionary keys to match exactly.

8. **`requiresOnDeviceRecognition = true` in SpeechTranscriber** is mandatory.
   If the device language doesn't have a downloaded on-device model, this throws.
   Catch this and show a Settings prompt: Settings → General → Offline Speech Recognition.

9. **Gemma 4 prompt format is strict.** `<start_of_turn>` / `<end_of_turn>` tokens
   must be exact. Any deviation causes incoherent output. Always use `PromptTemplates`.
   Never construct prompt strings inline.

10. **Model files are large.** E2B ≈ 1.5 GB, E4B ≈ 3 GB. The download in `ModelDownloader`
    uses a 1 MB buffer to avoid memory pressure. Do not load the entire response into
    memory. The current implementation streams bytes correctly — do not simplify it.
